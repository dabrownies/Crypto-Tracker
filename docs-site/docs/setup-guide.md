## Running the Web and Mobile Apps

### Web App
1. Install dependencies:
   ```bash
   npm install
   ```

2. Start Development Server:
   ```bash
   npm install
   ```

3. For Mobile App (React Native):
    ```bash
    npm install
    npm run android # For Android
    npm run ios     # For iOS (Mac required)
    ```

4. Open On Your Browser at:
```bash
   http://localhost:3000
```
